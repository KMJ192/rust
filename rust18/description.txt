트레잇
 - 다른종류의 추상화를 사용할 수 있도록 함
 - 타입들이 공통적으로 갖는 동작에 대하여 추상화 하도록 해줌
 - 러스트 컴파일러에게 특정한 타입이 갖고 다른 타입들과 함께 공유할 수도 있는 기능
 - Interface와 유사함
 - 서로 다른 타입에 대하 모두 동일한 Method로 호출
 - 한 Line에 하나의 Method 시그니처와 각 Line의 끝에 세미콜론을 갖도록 하여 여러 Method를 가질 수 있음
 - 한 타입은 여러 트레잇을 구현할 수 있음
 - 한 타입이 여러 트레잇을 구현할 떄 메서드 시그니처가 겹치면 Compile Error
