라이프 타임
 - 모든 참조자는 Life Time을 가짐
 - 해당 참조자가 유효한 Scope
 - Dangling 참조자(프로그램이 의도하지 않은 Data를 참조) 방지
 - 라이프타임 명시 문법
    &i32        // a reference
    &'a i32     // a reference with an explicit lifetime
    &'a mut i32 // a mutable reference with an explicit lifetime
 - 하나의 파라미터를 갖는 함수는 하나의 라이프타임 파라미터를 가짐,
   2개의 라이프타임 파라미터를 갖는 함수는 2개의 라이프타임 파라미터를 따로 가짐
 - 하나의 라이프타임 파라미터만 있다면 모든 출력 라이프타임 파라미터들에 대입
 - 여러개의 입력 라이프타임 파라미터가 있는데 메소드라서 그 중 하나가 &self 혹은 &mut self일 때 
   self의 라이프타임이 모든 출력 라이프 타임 파라미터에 대입됨
